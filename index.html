Projects

Field Efficiency and Route Efficiency Model

Project Summary: In this project, I developed a machine learning model to predict field and route efficiency for an agricultural field to optimize resource utilization and reduce costs.
Tools and technologies used:

  1. Geopandas and Pyspark to process geospatial data in three different data formats (CSV, XML, and SHP)
  2. PowerBI to create a dashboard to visualize the data and get insights for any selected field
  3. AWS S3, EMR, and Athena to store and process the data
  4. Scikit-learn to develop an XGboost based machine learning model to predict route and field efficiency
  5. Model interpretation approaches (Partial Dependence, Shapley Values, and Local Surrogate models) to visualize the internal working and validation of the blackbox model

Unit testing framework setup for Pyspark applications

Project Summary: Developed a unit testing framework for PySpark applications, which makes it easier to write and execute tests for complex PySpark code.

  1. Tools and technologies used:
    1. Python unit test module to create mock tests for core modules
    2. Pytest to create data validation tests
    3. Spark JVM to log messages at different levels
  2. Key Highlights:
    1. The framework divides unit testing into two parts: mock tests and data validation tests.
    2. Mock tests can be executed without a Spark environment or session, making them faster to run.
    3. Data validation tests use Pytest to test the output of Pyspark functions against expected results.
    4. The framework can be executed using Pytest to simplify unit testing.
    5. A logging utility class was developed to log messages at different levels using Spark JVM.
