Projects

Field Efficiency and Route Efficiency Model

Problem Statement: Like every other job, agricultural jobs like harvesting, spraying, spreading etc have limited availability of resources. As usage of these resources increases, cost to the farmer increases. Raven products help in automating the field jobs. However there is a need to predict efficiency of a field and route taken, that can be achieved for particular job.
Solution:
1. Processed Geospatial data multiple fields which was in three different data formats using Geopandas and Pyspark.
2. Created transformations in Pyspark to merge data from three different formats (CSV, XML and SHP).
3. Designed a PowerBI dashboard to view data and get insights for any selected field.
4. Developed end-to-end pipeline which extracts data from AWS S3, loads and processes the data in AWS EMR and stores the tranformed data into AWS Athena.
5. Built XGboost based Machine Learning model using scikit-learn which uses tranformed data and predicts route and field efficiency.
6. Generated a visualized report to show internal working and validation of the blackbox model using model interpretation approaches(Partial Dependence, Shapley Values and Local Surrogate models).
